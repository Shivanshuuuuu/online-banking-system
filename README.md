🏦 Online Banking System

This project is a full-stack Online Banking System developed using Spring Boot, MySQL, HTML, CSS, and JavaScript. It is designed to simulate the core functionalities of a real-world banking application, allowing users to securely register, log in, and manage their accounts through a simple and interactive interface. The application provides features such as user authentication, profile management, balance enquiry, and money transfer between users using their usernames.

The backend of the application is built using Spring Boot, where REST APIs are implemented to handle business logic and communication between the frontend and the database. MySQL is used for persistent data storage, ensuring that user details and account balances are maintained accurately. On the frontend side, HTML, CSS, and JavaScript are used to create a responsive and user-friendly interface, with seamless integration achieved using the Fetch API.

One of the key features of this system is the money transfer functionality, which includes proper validation checks such as verifying whether the receiver exists, ensuring sufficient balance before processing the transaction, and automatically updating the balance of both sender and receiver after a successful transfer. The project also includes profile update functionality where users can modify their personal details like name, email, and password.

This project helped in understanding how real-world applications manage user data, handle backend logic, and maintain communication between different layers of the system. It also provided hands-on experience with REST API development, database integration, and frontend-backend interaction.

Currently, further improvements are being planned, including the addition of transaction history, implementation of secure password encryption, and enhancement of the overall user interface and system design.
